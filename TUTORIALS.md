## 快速开始 (Quick Start)

### 第一次启动

1. **确保已安装依赖**
   ```bash
   pip install -r requirements.txt
   ```

2. **启动游戏启动器**
   ```bash
   python game_launcher.py
   ```

3. **选择游戏**
   - 使用数字键 1-5 快速选择
   - 或使用↑↓方向键浏览，按 Enter 确认

4. **调整摄像头位置**
   - 确保光线充足
   - 手部完全在摄像头视野内
   - 距离摄像头约 40-60cm

## 贪吃蛇游戏教程 (Snake Game Tutorial)

### 游戏目标
控制蛇吃食物，尽可能获得高分，避免撞到自己。

### 基本操作

1. **控制蛇的移动**
   - 伸出食指（其他手指可以握拳）
   - 将食指移动到你想让蛇头去的位置
   - 蛇头会跟随食指移动

2. **暂停游戏**
   - 方法1: 按键盘 `P` 键
   - 方法2: 做出"和平"手势 ✌️

3. **打开菜单**
   - 按键盘 `M` 键
   - 在菜单中可以调整难度、查看最高分等

### 难度级别

**简单模式**
- 蛇移动速度慢
- 适合初学者
- 分数倍率 1x

**中等模式**
- 蛇移动速度适中
- 平衡的挑战性
- 分数倍率 2x

**困难模式**
- 蛇移动速度快
- 适合高手
- 分数倍率 3x

### 得分技巧

1. **计划路线**: 提前规划移动路径，避免困住自己
2. **控制速度**: 不要移动太快，容易失控
3. **注意身体**: 当蛇变长时，小心不要撞到自己
4. **利用边缘**: 利用游戏区域边缘来引导蛇的移动

### 进阶技巧

- 在困难模式下尝试获得高分
- 尝试创造最长的蛇
- 挑战在不暂停的情况下完成游戏

## 水果切切乐教程 (Fruit Slicer Tutorial)

### 游戏目标
用手指滑动切割从底部弹出的水果，不要让水果掉落。

### 基本操作

1. **切割水果**
   - 伸出食指
   - 快速滑动手指经过水果
   - 水果会在手指路径上被切开

2. **生命值**
   - 开始时有3条生命
   - 每掉落一个水果失去1条生命
   - 生命值耗尽游戏结束

### 得分技巧

1. **连续切割**: 一次滑动切割多个水果可以获得更高分
2. **提前准备**: 观察水果出现位置，提前移动手指
3. **快速反应**: 游戏会越来越快，保持注意力集中
4. **流畅滑动**: 使用流畅的滑动动作而不是点击

### 进阶技巧

- 尝试连续切割5个以上水果
- 在高速模式下保持零失误
- 挑战最高连击记录

## 飞扬之手教程 (Flappy Hand Tutorial)

### 游戏目标
通过上下移动手控制小鸟飞行，避开管道障碍，尽可能飞得更远。

### 基本操作

1. **控制小鸟高度**
   - 将手向上移动，小鸟上升
   - 将手向下移动，小鸟下降
   - 保持手在摄像头视野内

2. **通过管道**
   - 对准管道中间的空隙
   - 保持平稳飞行
   - 每通过一个管道得1分

### 飞行技巧

1. **平稳移动**: 避免突然移动手，保持平稳
2. **提前调整**: 看到管道时提前调整高度
3. **找到节奏**: 找到适合自己的控制节奏
4. **保持冷静**: 不要紧张，放松操作

### 进阶技巧

- 尝试不看手的位置，只看屏幕
- 挑战连续通过20个管道
- 尝试用非惯用手控制

## 石头剪刀布教程 (Rock Paper Scissors Tutorial)

### 游戏目标
使用手势与电脑进行石头剪刀布对战，赢得更多回合。

### 手势说明

**石头** ✊
- 握紧拳头
- 所有手指收起

**布** ✋
- 张开手掌
- 五指伸直分开

**剪刀** ✌️
- 伸出食指和中指
- 其他手指收起

### 游戏流程

1. **选择手势**
   - 做出你想要的手势
   - 游戏会自动识别并开始倒计时

2. **倒计时**
   - 屏幕显示 3、2、1 倒计时
   - 保持手势稳定

3. **结果显示**
   - 显示你和电脑的选择
   - 显示谁赢了
   - 自动更新比分

4. **继续游戏**
   - 结果显示后自动返回等待状态
   - 可以继续做新的手势开始下一回合

### 策略技巧

1. **随机选择**: 电脑是随机选择的，没有固定模式
2. **连胜记录**: 尝试创造连胜记录
3. **快速游戏**: 熟练后可以快速连续对战

## 空中绘画教程 (Air Drawing Tutorial)

### 游戏目标
使用手指在空中绘画，创作艺术作品。

### 基本操作

1. **开始绘画**
   - 伸出食指（只有食指）
   - 移动手指进行绘画
   - 绘画轨迹会显示在屏幕上

2. **停止绘画**
   - 张开整个手掌
   - 或收起食指

### 工具使用

**画笔工具**
- 默认工具
- 用于绘画线条
- 可选择颜色和粗细

**橡皮擦工具**
- 按 `T` 键切换
- 用于擦除绘画
- 可调整大小

### 功能键说明

| 按键 | 功能 |
|------|------|
| C | 切换颜色 |
| T | 切换工具（画笔/橡皮擦） |
| + | 增加画笔/橡皮擦大小 |
| - | 减少画笔/橡皮擦大小 |
| U | 撤销上一步 |
| X | 清除整个画布 |
| H | 显示/隐藏帮助信息 |
| P | 显示/隐藏调色板 |

### 颜色选择

游戏提供8种颜色：
1. 红色
2. 绿色
3. 蓝色
4. 黄色
5. 洋红色
6. 青色
7. 黑色
8. 灰色

按 `P` 键显示调色板，按数字键 1-8 快速选择颜色。

### 绘画技巧

1. **稳定手指**: 保持手指稳定可以画出平滑的线条
2. **调整距离**: 适当调整与摄像头的距离
3. **使用橡皮擦**: 不满意的部分可以用橡皮擦修改
4. **善用撤销**: 撤销功能可以恢复到前一步
5. **分层绘画**: 先画大框架，再添加细节

### 创意建议

- 尝试画简单的图形（圆形、方形、星星）
- 画一幅风景画
- 写字或签名
- 创作抽象艺术
- 画你的梦想

## 手势识别指南 (Gesture Recognition Guide)

### 支持的手势

**指向** (Point)
- 只伸出食指
- 用途: 绘画、指向控制

**和平手势** (Peace) ✌️
- 伸出食指和中指
- 用途: 暂停菜单、剪刀

**张开手掌** (Open Palm) ✋
- 五指张开
- 用途: 布、停止绘画

**拳头** (Fist) ✊
- 握紧拳头
- 用途: 石头

**捏合** (Pinch)
- 拇指和食指靠近
- 用途: 特殊功能

**竖起大拇指** (Thumbs Up) 👍
- 只竖起大拇指
- 用途: 确认、点赞

### 手势识别最佳实践

1. **光线充足**: 确保环境光线良好
2. **背景简单**: 避免复杂的背景干扰
3. **手部清晰**: 整个手掌在摄像头视野内
4. **动作清晰**: 手势要明确，不要模棱两可
5. **保持稳定**: 做手势时保持手部稳定1-2秒

### 识别问题排查

**手势无法识别**
- 检查光线是否充足
- 确认手部完全在视野内
- 尝试调整手与摄像头的距离
- 确保手势动作清晰明确

**识别不准确**
- 减少背景干扰
- 保持手部稳定
- 调整检测置信度（在代码中）

## 常见问题 (FAQ)

### 游戏相关

**Q: 如何切换游戏？**
A: 按 ESC 键返回游戏选择菜单，然后选择新游戏。

**Q: 游戏数据保存在哪里？**
A: 游戏数据保存在 `game_data.json` 文件中，包括最高分和设置。

**Q: 可以删除历史记录吗？**
A: 可以删除 `game_data.json` 文件来重置所有数据。

**Q: 游戏卡顿怎么办？**
A: 尝试关闭其他占用CPU的程序，或降低摄像头分辨率。

### 手势识别

**Q: 手势识别延迟怎么办？**
A: 确保光线充足，减少背景干扰，或调整摄像头位置。

**Q: 某些手势无法识别？**
A: 尝试更明确地做手势，保持手势稳定1-2秒。

**Q: 可以左手玩吗？**
A: 可以，系统支持左手和右手。

### 技术问题

**Q: 需要什么配置的电脑？**
A: 需要摄像头、Python 3.7+，以及中等配置的CPU。

**Q: 支持哪些操作系统？**
A: 支持 Windows、macOS 和 Linux。

**Q: 可以用外置摄像头吗？**
A: 可以，如果默认摄像头不对，修改代码中的摄像头索引。

### 游戏玩法

**Q: 贪吃蛇怎么得高分？**
A: 选择困难模式，计划好路线，避免撞到自己。

**Q: 水果切切乐怎么连击？**
A: 用流畅的滑动动作，一次切过多个水果。

**Q: 空中绘画的作品可以保存吗？**
A: 目前不支持自动保存，但可以截屏保存。

**Q: 石头剪刀布有技巧吗？**
A: 电脑是随机的，没有固定规律，尽情享受对战乐趣。
