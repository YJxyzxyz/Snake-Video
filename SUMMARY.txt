WHAT WAS IMPLEMENTED / 实现内容
================================================================================

A complete hand-gesture controlled Snake game that uses a webcam to detect
hand movements and allows users to control the snake using their index finger.

HOW TO USE / 使用方法
================================================================================

1. Clone the repository:
   git clone https://github.com/YJxyzxyz/Snake-Video.git
   cd Snake-Video

2. Install dependencies:
   pip install -r requirements.txt

3. Run the game:
   python start.py  (recommended - includes checks)
   OR
   python main.py   (direct start)

4. Play:
   - Move your hand in front of the camera
   - Point your index finger to control the snake
   - Collect red food to grow and score
   - Press 'R' to restart, 'Q' to quit
