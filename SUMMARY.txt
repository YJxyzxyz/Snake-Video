HOW TO USE / 使用方法
================================================================================

1. Clone the repository:
   git clone https://github.com/YJxyzxyz/Snake-Video.git
   cd Snake-Video

2. Install dependencies:
   pip install -r requirements.txt

3. Run the game launcher (recommended):
   python game_launcher.py
   
   OR run individual games:
   python main.py          (Snake Game)
   python start.py         (Snake with system checks)

4. Select and play:
   - Choose a game from the menu (1-5)
   - Follow on-screen instructions
   - Use gestures to control
   - Press 'Q' to quit, 'ESC' for menu
